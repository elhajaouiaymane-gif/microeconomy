SECTION 1 — Définitions (Q1-Q8)
1. PM (productivité moyenne)
→ PML = Q / L
→ Trap: confusion avec Pm (ΔQ/ΔL)
⇒ PM = total/total, Pm = variation
2. Cm (coût marginal)
→ Cm = dCT/dq (dérivée)
→ Trap: confondre avec CM = CT/q
⇒ Cm = dérivée, CM = division
3. TMST (taux marginal de substitution)
→ TMST = PmL / PmK
→ Trap: inverser les productivités
⇒ Travail/capital : PmL/PmK
4. Offre CPP
→ p = Cm (règle d'or de la CPP)
→ Trap: utiliser p = CM
⇒ En CPP, produire tant que p=Cm
5. Profit
→ π = p·q − CT(q)
→ Trap: oublier CT (juste RT)
⇒ Profit = RT − CT
6. Équilibre marché
→ QO = QD
→ Trap: croire QO > QD = équilibre
⇒ Offre = Demande
7. Seuil de fermeture
→ p = min CVM (en dessous → q=0)
→ Trap: confondre avec seuil rentabilité
⇒ Fermeture = min CVM
8. Décision production CT
→ Comparer p à min CVM et min CM
→ Trap: fermer dès que p < min CM
⇒ On produit tant que p > min CVM
SECTION 2 — Production + Coûts / Offre (Q9-Q16)
9. Calcul PM
→ Q/L avec chiffres donnés
→ Ex: Q=100, L=5 → PM=20
⇒ Q divisé par L
10. Calcul Pm
→ ΔQ/ΔL = (Q2−Q1)/(L2−L1)
→ Ex: Q:80→96, L:4→5 → Pm=16
⇒ Variation de Q sur variation de L
11. TMST Cobb-Douglas
→ Q = A·LαKβ → TMST = (α/β)·(K/L)
→ Ex: α=0,7 ; β=0,3 → TMST = 14K/6L
⇒ αK / βL
12. Rendements déchelle
→ α+β = 1 → constants ; >1 → croissants ; <1 → décroissants
→ Ex: 0,6+0,4=1 → constants
⇒ Somme des exposants
13. Optimum producteur
→ TMST = PL/PK → isole K=f(L)
→ Ex: 3K/2L=6/2=3 → K=2L
⇒ TMST = rapport prix → K = f(L)
14. Identifier CF et CVM
→ CF = constante de CT ; CVM = (CT−CF)/q
→ Ex: CT=3q2+60q+7000 → CF=7000, CVM=3q+60
⇒ CF = constante, CVM = partie variable / q
15. q* par p=Cm
→ Cm = dCT/dq → p=Cm → q*
→ Ex: CT=q2+6q+100, p=80 → Cm=2q+6, 80=2q+6, q=37
⇒ Dériver → p=Cm → isoler q
16. Profit multi-step
→ CT → Cm → p=Cm → q* → π
→ Ex: CT=3y2+60y+7000, p=390 → y=55, π=2075
⇒ 4 étapes : Cm → q* → CT(q*) → RT−CT
SECTION 3 — Marché CPP (Q17-Q24)
17. Équilibre QO=QD → p*
→ Résoudre QO = QD pour p
→ Ex: 5p−50 = 2050−20p → 25p=2100 → p=84
⇒ Les p d'un côté, constantes de l'autre
18. Q* à l'équilibre
→ Remplacer p* dans QO ou QD
→ Ex: p=84 dans QO=5p−50 → Q*=370
⇒ Q* = QO(p*) = QD(p*)
19. q* individuelle
→ p=Cm → q(p) → q*=q(p*)
→ Ex: q=(p−10)/2, p=84 → q=37
⇒ Appliquer p* dans la fonction d'offre
20. Nombre d'entreprises
→ N = Q* / q*
→ Ex: Q*=370, q*=37 → N=10
⇒ N = total / individuel
21. Seuil fermeture (calcul)
→ min CVM : dériver CVM, poser =0
→ Ex: CVM=y+6 → min=6 à y=0
⇒ Fermeture = min CVM = 6
22. Seuil rentabilité (calcul)
→ min CM : dériver CM, poser Cm=CM
→ Ex: CM=y+6+100/y → min=26 à y=10
⇒ Rentabilité = min CM = 26
23. Marché multi-step complet
→ QO=QD → p* → Q* → q* → N (5 étapes)
→ Ex: QO=5p−20, QD=310−6p, Cm=2q+4 → p=30, Q=130, q=13, N=10
⇒ QO=QD → p → Q ; p=Cm → q ; N = Q/q
24. Relation Cm / CM
→ Cm < CM → CM ↓ ; Cm > CM → CM ↑
→ Cm = CM au minimum du CM
⇒ Le marginal tire le moyen vers lui

16 formules TD

CM = CT / q
CVM = CV / q
Cm = dCT / dq
CPP : p = Cm
π = p·q − CT(q)
QO = QD
N = Q / q
Fermeture = min CVM
Rentabilité = min CM
Cm < CM → CM ↓

Optimum : TMST = PL/PK

16 Traps TD

1. PM vs Pm
PM = Q/L (total), Pm = ΔQ/ΔL (variation)
Règle: «moyenne» = total/total, «marginale» = variation
2. Dérivée CT
q2 → q (faux) ; vrai : q2 → 2q
Règle: puissance descend en multipliant
3. CM vs Cm
Trap: p = CM (faux). Vrai: p = Cm en CPP
Règle: CM = CT/q (moyen), Cm = dCT/dq (marginal)
4. Oubli CF dans CM
CM = 3q+60 (oubli 7000/q). Vrai: diviser chaque terme de CT par q
5. CF dans CVM
Enlever CF avant de diviser par q. CF ne va pas dans CVM.
6. Signe offre inverse
p=2q+6 → q=2p+6 (faux). Vrai: q = 0,5p−3. Isoler q correctement.
7. TMST inversé
Trap: TMST = PmK/PmL. Vrai: TMST = PmL/PmK = L/capital
8. Confusion Q* et q*
N = q*/Q* (faux). Vrai: N = Q*/q*. Q* = marché total, q* = une firme
9. Seuil fermeture vs rentabilité
Fermeture = min CVM, rentabilité = min CM. CVM < CM toujours.
10. Profit oubli CT
Trap: π = p·q (RT seulement). Vrai: π = RT − CT(q)
11. Signe équation marché
5p−50 = 2050−20p → 5p−20p = 2050−50 (faux). Les signes changent !
12. Oubli étape profit
Trap: π = p·q sans calculer CT. Il faut CT(q*) après avoir trouvé q* par p=Cm.
13. Décision CT
Trap: si p < min CM → fermer. Vrai: on ferme que si p < min CVM.
14. Parenthèses offre
q = (p−10)/2 → ne pas oublier les parenthèses: (84−10)/2 = 37, pas 84/2−10 = 32
15. Confondre CT et Cm
CT n'est pas Cm. Dériver avant d'appliquer p=Cm.
16. TMST à l'optimum
Égaler TMST au rapport des prix, puis résoudre K en fonction de L.
