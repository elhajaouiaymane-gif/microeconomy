TD1 — Production (Q1-Q8)
1. Isoquante
Courbe des combinaisons (L,K) de même production Q. Décroissante et convexe. Pente = −TMST.
⇒ Isoquante = Q constant, pente négative, convexe
2. TMST
TMST = PmL/PmK = pente de l'isoquante. Mesure combien de K on remplace par 1L à Q constant.
Quand L augmente, TMST diminue → isoquante convexe.
⇒ TMST = pente isoquante = PmL/PmK
3. Isocoût
CT = PL·L + PK·K. Pente = −PL/PK.
Si PL↑ → pente plus raide. Si PK↓ → pente plus plate.
⇒ Isocoût = même CT, pente = −PL/PK
4. Optimum producteur
TMST = PL/PK → tangence isoquante/isocoût.
La productivité marginale par euro est égalisée entre inputs.
⇒ Optimum = TMST = PL/PK (tangence)
5. PM (productivité moyenne)
PML = Q / L. Production totale divisée par les travailleurs.
Trap: confondre avec Pm = ΔQ/ΔL (marginale).
⇒ PM = total/total, Pm = variation
6. Pm (productivité marginale)
PmL = ΔQ/ΔL. Production ajoutée par le dernier travailleur.
Ex: Q:80→96, L:4→5 → Pm = 16.
⇒ Pm = variation de Q / variation de L
7. TMST Cobb-Douglas
Q = A·LαKβ → TMST = (α/β)·(K/L).
Ex: α=0,7, β=0,3 → TMST = 14K/6L.
Rendements: α+β = 1 → constants ; >1 → croissants ; <1 → décroissants.
⇒ TMST = (αK)/(βL) ; rendements = α+β
8. Optimum K = f(L)
Égaler TMST au rapport des prix. Résoudre K en fonction de L.
Ex: TMST=3K/2L, PL=6, PK=2 → 3K/2L=3 → K=2L.
⇒ TMST = PL/PK → isole K = f(L)
TD2 — Coûts et Offre (Q9-Q16)
9. CM vs Cm
CM = CT/q (moyen). Cm = dCT/dq (marginal). Ne pas les confondre !
Trap: utiliser p = CM au lieu de p = Cm.
⇒ CM = CT/q ; Cm = dCT/dq
10. Seuil fermeture
p = min CVM. En dessous → fermer à court terme (q=0).
Ne pas confondre avec seuil de rentabilité (min CM).
⇒ Fermeture = min CVM
11. Seuil rentabilité
p = min CM. Au-dessus → profit ≥ 0.
Trouvé par Cm = CM (ou dérivée de CM = 0).
⇒ Rentabilité = min CM
12. Décision court terme
p < min CVM → fermer. min CVM < p < min CM → produire avec perte. p > min CM → profit.
Trap: fermer dès que p < min CM (faux !).
⇒ On produit tant que p > min CVM
13. CF et CV
CT(q)→ CF = terme constant. CV = CT − CF.
Ex: CT=3q2+60q+7000 → CF=7000, CV=3q2+60q.
⇒ CF = constante, CV = CT − CF
14. CM, CVM, Cm calcul
CM = CT/q ; CVM = CV/q ; Cm = dCT/dq.
Ex: CT=3q2+60q+7000 → CM=3q+60+7000/q, CVM=3q+60, Cm=6q+60.
⇒ CM = division, CVM = CV/q, Cm = dérivée
15. q* par p = Cm
Dériver CT → Cm. Poser p=Cm. Résoudre q.
Ex: CT=q2+6q+100, p=80 → Cm=2q+6, 80=2q+6 → q=37.
⇒ p = Cm → q*
16. Profit multi-step
4 étapes: (1) Cm = dCT/dq (2) p=Cm → q* (3) CT(q*) (4) π = p·q − CT(q*).
Ex: CT=3y2+60y+7000, p=390 → y=55, CT=19375, π=2075.
⇒ Cm → q* → CT → π
TD3 — Marché CPP (Q17-Q24)
17. Équilibre marché
QO = QD. Résoudre l'égalité pour trouver p*.
Signes: déplacer les p d'un côté, les constantes de l'autre, en changeant les signes.
⇒ QO = QD → p*
18. Q*
Remplacer p* dans QO ou QD. Les deux donnent le même résultat.
Ex: p*=84, QO=5p−50 → Q*=370.
⇒ Q* = QO(p*) = QD(p*)
19. q* individu
Appliquer p* dans la fonction d'offre individuelle q(p).
Ex: q=(p−10)/2, p=84 → q=37. Attention aux parenthèses !
⇒ q* = q(p*)
20. N = Q/q
Nombre d'entreprises = quantité totale / quantité par firme.
Ne pas confondre Q* (marché) et q* (firme).
⇒ N = Q* / q*
21. Marché complet
5 étapes: QO=QD → p* → Q* → p=Cm → q* → N=Q*/q*.
Ex: QO=5p−20, QD=310−6p, Cm=2q+4 → p=30, Q=130, q=13, N=10.
⇒ QO=QD → p → Q ; p=Cm → q ; N=Q/q
22. Fermeture calcul
min CVM. Idéalement dériver CVM et poser = 0.
Ex: CVM=y+6 → min=6 à y=0.
⇒ min CVM = seuil fermeture
23. Rentabilité calcul
min CM. Résoudre Cm=CM ou dériver CM=0.
Ex: CM=y+6+100/y → Cm=CM → y=10 → CM=26.
⇒ min CM = seuil rentabilité
24. Relation Cm/CM
Cm < CM → CM ↓ (marginal tire le moyen vers le bas).
Cm > CM → CM ↑. Cm = CM au minimum du CM.
⇒ Le marginal tire le moyen

20 Traps TD

— TD1 Production —
1. PM vs Pm
PM = Q/L (moyenne). Pm = ΔQ/ΔL (marginale). Ne pas les confondre.
2. TMST inversé
TMST = PmK/PmL (faux). Vrai: TMST = PmL/PmK.
3. Isoquante = même coût
Trap: croire isoquante = même coût. Non, isoquante = même production Q.
4. Isoquante croissante
Trap: croire isoquante croissante. Faux: si L↑ alors K↓ à Q constant.
5. Pente isocoût = PL/PK
Trap: oublier le signe. Pente = -PL/PK (négative).
6. Confusion optimum
Trap: optimum = PM=PL ou Pm=PK. Vrai: TMST = PL/PK.
— TD2 Coûts —
7. Dérivée CT
q2 → q (faux). Vrai: q2 → 2q. La puissance descend.
8. CM vs Cm
Trap: p = CM. Vrai: p = Cm en CPP.
9. Oubli CF dans CM
CM = 3q+60 (oubli 7000/q). Divise chaque terme de CT par q.
10. CF dans CVM
CVM = (3q2+60q+7000)/q (faux). Enlever CF d'abord.
11. Signe offre inverse
p=2q+6 → q=2p+6 (faux). Vrai: q = 0,5p−3.
12. Profit oubli CT
π = p·q (RT seulement). Vrai: π = p·q − CT(q).
13. Oubli étape multi-step
Ne pas oublier de calculer CT(q*) avant de soustraire de RT.
14. Décision CT erronée
Trap: p < min CM → fermer. Vrai: on ferme si p < min CVM.
15. Seuil fermeture/rentabilité
Trap: les confondre. Fermeture = min CVM ; rentabilité = min CM.
— TD3 Marché —
16. Signes équation
5p−50=2050−20p → 5p−20p=2050−50 (faux). Les signes changent.
17. Confusion Q* et q*
N = q*/Q* (faux). Vrai: N = Q*/q*. Marché vs firme.
18. Parenthèses offre
q=(p−10)/2. Avec p=84: (84−10)/2=37, pas 84/2−10=32.
19. Oubli −50 dans QO
Q*=5×84=420. Faux: Q*=5×84−50=370.
20. Confondre CT et Cm
Trap: poser p=CT au lieu de p=Cm. Il faut dériver d'abord.
