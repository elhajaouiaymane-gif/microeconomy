Les 24 patterns TD — calculs ET concepts

Chaque pattern = un type de question qui tombe à l'examen.

SECTION 1 — Définitions + Concepts (Q1-Q8)
1. PM (productivité moyenne)
→ PML = Q / L (moyenne)
→ Trap: confondre avec Pm = ΔQ/ΔL (marginale)
⇒ PM = total/total, Pm = variation
2. Cm (coût marginal)
→ Cm = dCT/dq (dérivée), pas CT/q
→ Trap: CM = CT/q est la moyenne, Cm est le coût de la dernière unité
⇒ Cm = dérivée, CM = division
3. TMST — interprétation
→ TMST = PmL/PmK = pente de l'isoquante
→ Mesure combien de K on remplace par 1L à Q constant
⇒ TMST = pente isoquante = substitution à Q constant
4. Isoquante
→ Courbe décroissante : si L↑ alors K↓ pour Q fixe
→ convexe (TMST décroissant). JAMAIS croissante
⇒ Isoquante = même Q, pente négative, convexe
5. Isocoût
→ CT = PL·L + PK·K, pente = −PL/PK
→ Si PL↑ → pente plus raide. Si PK↓ → pente plus plate
⇒ Isocoût = même CT, pente = −PL/PK
6. Optimum producteur
→ TMST = PL/PK (tangence isoquante/isocoût)
→ Productivité marginale par euro égalisée entre inputs
⇒ Optimum = isoquante tangente à isocoût
7. Offre CPP
→ p = Cm → maximise le profit
→ Trap: utiliser p = CM (faux en CPP)
⇒ En CPP : produire tant que p = Cm
8. Décision court terme
→ p < min CVM → fermer. min CVM < p < min CM → perte
→ Ne pas fermer dès que p < min CM !
⇒ On produit tant que p > min CVM
SECTION 2 — Production + Coûts / Offre (Q9-Q16)
9. PM calcul
→ Q/L avec valeurs numériques
→ Ex: Q=100, L=5 → PM = 20
⇒ Q divisé par L
10. Pm calcul
→ ΔQ/ΔL = (Q2−Q1)/(L2−L1)
→ Ex: Q:80→96, L:4→5 → Pm = 16
⇒ Variation de Q / variation de L
11. TMST Cobb-Douglas
→ Q = A·LαKβ → TMST = (α/β)·(K/L)
→ Ex: α=0,7 ; β=0,3 → TMST = 14K/6L
⇒ (αK) / (βL)
12. Rendements déchelle
→ α+β = 1 → constants ; >1 → croissants ; <1 → décroissants
→ Ex: 0,6+0,4=1 → constants
⇒ Somme des exposants
13. Optimum producteur
→ TMST = PL/PK → K = f(L)
→ Ex: TMST=3K/2L, PL=6, PK=2 → K=2L
⇒ Égaler TMST au rapport des prix
14. CF, CVM, offre
→ CF = constante ; CVM = CV/q ; p=Cm → q(p)
→ Ne pas mettre CF dans CVM
⇒ CF = constante, CVM = partie variable / q
15. q* par p=Cm
→ Dériver CT → Cm ; poser p=Cm → résoudre q
→ Ex: CT=q2+6q+100, p=80 → q=37
⇒ Dériver → p=Cm → isoler q
16. Profit multi-step
→ CT → Cm → q* → CT(q*) → π
→ Ex: CT=3y2+60y+7000, p=390 → y=55, π=2075
⇒ 4 étapes : Cm → q* → CT(q*) → RT−CT
SECTION 3 — Marché CPP (Q17-Q24)
17. Équilibre QO=QD → p*
→ Résoudre QO = QD + signes corrects
→ Ex: 5p−50 = 2050−20p → p=84
⇒ Les p d'un côté, constantes de l'autre
18. Q* à l'équilibre
→ Remplacer p* dans QO ou QD
→ Q* = QO(p*) = QD(p*)
⇒ Q* = QO(p*)
19. q* individuelle
→ q(p) donnée, remplacer p*
→ Ex: q=(p−10)/2, p=84 → q=37
⇒ Attention aux parenthèses
20. Nombre d'entreprises
→ N = Q* / q*
→ Ex: Q*=370, q*=37 → N=10
⇒ N = total / individuel
21. Seuil fermeture
→ min CVM : dériver CVM, trouver minimum
→ Ex: CVM=y+6 → min=6 à y=0
⇒ Fermeture = min CVM
22. Seuil rentabilité
→ min CM : Cm=CM → résoudre q puis CM
→ Ex: CM=y+6+100/y → min=26 à y=10
⇒ Rentabilité = min CM
23. Marché complet
→ QO=QD → p* → Q* → q* → N (5 étapes)
→ Ex: QO=5p−20, QD=310−6p, Cm=2q+4 → p=30, Q=130, q=13, N=10
⇒ QO=QD → p → Q ; p=Cm → q ; N=Q/q
24. Relation Cm/CM + équilibre firme
→ Cm < CM → CM ↓ ; Cm > CM → CM ↑
→ Cm = CM au min du CM (profit nul si p=CM)
⇒ Le marginal tire le moyen

18 Traps TD

1. PM vs Pm
PM = Q/L (total), Pm = ΔQ/ΔL (variation). Ne pas les confondre.
2. Dérivée CT
q2 → q (faux) ; vrai : q2 → 2q. La puissance descend.
3. CM vs Cm
Trap: p = CM (faux). Vrai: p = Cm. CM = CT/q, Cm = dCT/dq.
4. Oubli CF dans CM
CM = 3q+60 (oubli 7000/q). Divisé chaque terme de CT par q.
5. CF dans CVM
Enlever CF d'abord. Le CF constant ne va pas dans le CVM.
6. Signe offre inverse
p=2q+6 → q=2p+6 (faux). Vrai: q = 0,5p−3. Isoler q !
7. TMST inversé
TMST = PmK/PmL (faux). Vrai: TMST = PmL/PmK = travail/capital.
8. Confusion Q* et q*
N = q*/Q* (faux). Vrai: N = Q*/q*. Marché / firme.
9. Seuil fermeture vs rentabilité
Fermeture = min CVM. Rentabilité = min CM. CVM < CM toujours.
10. Profit oubli CT
π = p·q (RT seulement). Vrai: π = RT − CT(q).
11. Signes équation marché
5p−50 = 2050−20p → 5p+20p = 2050+50. Les signes changent !
12. Oubli étape multi-step
Ne pas oublier de calculer CT(q*) après avoir trouvé q*.
13. Décision CT erronée
Trap: p < min CM → fermer (faux). On ferme si p < min CVM.
14. Parenthèses offre
q = (p−10)/2 → (84−10)/2 = 37, PAS 84/2−10 = 32.
15. Isoquante = croissante
Trap: croire isoquante croissante. Faux: elle est décroissante (si L↑ K↓).
16. Isoquante = même coût
Trap: confondre isoquante (production) et isocoût (coût). Deux concepts différents.
17. Pente isocoût = PL/PK
Trap: pente isocoût = PL/PK (faux). Vrai: pente = −PL/PK (signe négatif).
18. Confondre CT et Cm
CT n'est pas la dérivée. Il faut dériver CT pour trouver Cm avant p=Cm.
