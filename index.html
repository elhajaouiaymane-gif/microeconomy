Les 16 patterns TD — Reconnais et réponds vite

Chaque pattern = un type d'exercice TD.

1. PM (productivité moyenne)
→ Pattern: PML = Q / L
→ Exemple: Q=100, L=5 → PM = 20
→ Trap: confusion avec Pm
⇒ Shortcut: PM = total/total, Pm = variation
2. Pm (productivité marginale)
→ Pattern: PmL = ΔQ / ΔL
→ Exemple: Q:80→96, L:4→5 → Pm = 16
→ Trap: oublier de diviser par ΔL
⇒ Shortcut: variation de Q / variation de L
3. TMST
→ Pattern: TMST = PmL / PmK
→ Exemple: Cobb-Douglas Q=20L0,7K0,3 → TMST = 14K/6L
→ Trap: inverser PmL/PmK ou confondre K et L
⇒ Shortcut: PmL/PmK, les L et K s\'annulent
4. Rendements d'échelle
→ Pattern: Cobb-Douglas α+β = degré d'homogénéité
→ Exemple: α+β=1 → constants ; >1 → croissants ; <1 → décroissants
⇒ Shortcut: somme des exposants
5. Optimum producteur
→ Pattern: TMST = PL/PK
→ Exemple: TMST=3K/2L, PL=6, PK=2 → 3K/2L=3 → K=2L
⇒ Shortcut: TMST = rapport des prix, puis isole K=f(L)
6. CF (coût fixe)
→ Pattern: CF = terme constant dans CT(q)
→ Exemple: CT = q2+6q+100 → CF = 100
⇒ Shortcut: CF = constante, ne dépend pas de q
7. Cm (coût marginal)
→ Pattern: Cm = dCT/dq (dérivée)
→ Exemple: CT = q2+6q+100 → Cm = 2q+6
→ Trap: oublier le 6q → 6
⇒ Shortcut: q2→2q, q→coeff, constante→0
8. CVM (coût variable moyen)
→ Pattern: CVM = CV / q
→ Exemple: CT=3y2+60y+7000 → CV=3y2+60y → CVM=3y+60
→ Trap: oublier que CF n'est pas dans CV
⇒ Shortcut: CT − CF = CV, puis diviser par q
9. CM (coût moyen)
→ Pattern: CM = CT / q
→ Exemple: CT=3q2+60q+7000 → CM=3q+60+7000/q
→ Trap: oublier de diviser CF par q
⇒ Shortcut: diviser chaque terme de CT par q
10. Offre (CPP)
→ Pattern: p = Cm → isoler q(p)
→ Exemple: Cm=2q+6 → p=2q+6 → q=0,5p−3
→ Trap: utiliser p=CM au lieu de p=Cm
⇒ Shortcut: dériver CT, poser p=Cm, isoler q
11. Profit multi-step
→ Pattern: CT → Cm → q* → π
→ Formule: Étape 1: Cm = dCT/dq. Étape 2: p = Cm → q*. Étape 3: π = p·q − CT(q)
→ Exemple: CT=y²+10y+144, p=70 → Cm=2y+10 → 70=2y+10 → y=30 → π=756
→ Trap: oublier CF dans CT ou oublier une étape
⇒ Shortcut: Deriver → p=Cm → q* → RT − CT
12. Marché multi-step
→ Pattern: QO=QD → p* → Q* → q* → N
→ Formule: Étape 1: QO=QD → p*. Étape 2: Q*=QO(p*). Étape 3: p=Cm → q*. Étape 4: N=Q*/q*
→ Exemple: QO=3p−10, QD=90−2p, Cm=2q+10 → p=20, Q=50, q=5, N=10
→ Trap: oublier -10 dans QO, ou dérivée fausse
⇒ Shortcut: QO=QD → p ; p=Cm → q ; N = Q/q
13. Relation Cm / CM
→ Pattern: Cm < CM → CM ↓ ; Cm > CM → CM ↑
→ Règle: Le marginal tire le moyen vers lui
→ Exemple: Cm=2q+6, CM=q+6+100/q → à q=10: Cm=26, CM=26 → Cm=CM au min CM
→ Trap: croire que CM est toujours au-dessus
⇒ Shortcut: marginal < moyen → moyen baisse
14. Seuil de fermeture
→ Pattern: p < min CVM → q=0 (fermeture court terme)
→ Formule: min CVM = prix plancher. Si p en dessous, produire 0
→ Exemple: CVM = y+6 → min CVM = 6. Si p=5 < 6 → q=0
→ Trap: confondre avec seuil de rentabilité (min CM)
⇒ Shortcut: Fermeture = min CVM (pas CM). Toujours min CVM < min CM
15. Seuil de rentabilité
→ Pattern: p ≥ min CM → profit ≥ 0
→ Formule: min CM = prix de rentabilité. Au-dessus = profit positif
→ Exemple: CM = y+6+100/y → min CM = 26 à y=10. p=30 > 26 → profit > 0
→ Trap: donner q (10) au lieu de p (26)
⇒ Shortcut: Rentabilité = min CM. Profit nul à ce prix exact
16. Décision de production
→ Pattern: Comparer p aux seuils pour décider
→ Règle: p < min CVM → q=0 (fermer). min CVM < p < min CM → produire mais perte. p > min CM → profit > 0
→ Exemple: min CVM=6, min CM=26. p=10 → produire avec perte. p=30 → profit
→ Trap: fermer dès que p < min CM (faux: on produit tant que p > min CVM)
⇒ Shortcut: Court terme: comparer à min CVM. Long terme: comparer à min CM
Les 14 formules à savoir par cœur :
1. PM = Q/L   2. Pm = ΔQ/ΔL   3. TMST = PmL/PmK   4. α+β = degré homog.   5. CT = CF + CV   6. CM = CT/q   7. CVM = CV/q   8. Cm = dCT/dq   9. p = Cm (offre)   10. π = p·q − CT(q)   11. QO = QD   12. N = Q/q   13. Fermeture = min CVM   14. Rentabilité = min CM

13 Traps TD

1. PM vs Pm
Trap: PM = ΔQ/ΔL
Vrai: PM = Q/L (total), Pm = ΔQ/ΔL (variation)
Règle: «moyenne» = total/total, «marginale» = variation
2. Dérivée CT
Trap: q2 → q
Vrai: q2 → 2q
Règle: puissance descend en multipliant
3. CM vs Cm
Trap: p = CM
Vrai: p = Cm en CPP
Règle: CM = CT/q (moyen), Cm = dCT/dq (marginal)
4. Oubli CF dans CM
Trap: CM = 3q+60 (oubli 7000/q)
Vrai: CM = 3q+60+7000/q
Règle: diviser CHAQUE terme de CT par q
5. CF dans CVM
Trap: CVM = (3q2+60q+7000)/q
Vrai: CVM = (3q2+60q)/q = 3q+60
Règle: enlever CF (constante) d'abord
6. Signe offre inverse
Trap: p=2q+6 → q=2p+6
Vrai: q = 0,5p−3
Règle: isoler q : p→q = (p−b)/2
7. TMST inversé
Trap: TMST = PmK/PmL
Vrai: TMST = PmL/PmK
Règle: travail/capital, toujours PmL/PmK
8. Confusion Q* et q*
Trap: N = q*/Q*
Vrai: N = Q*/q*
Règle: Q* = marché total, q* = une firme
9. Seuil fermeture vs rentabilité
Trap: seuil fermeture = min CM
Vrai: fermeture = min CVM, rentabilité = min CM
Règle: CVM < CM (car CF non couvert)
10. Profit oubli CT
Trap: π = p·q (oubli de CT)
Vrai: π = RT − CT = p·q − CT(q)
Règle: recette totale moins coût total
11. Oubli d'étape profit multi-step
Trap: π = p·q (RT) sans calculer CT
Vrai: π = RT − CT. Il faut d'abord trouver q* par p=Cm, puis calculer CT(q*)
Règle: q* dans Cm → CT dans CT(q*) → π = RT − CT
12. Signe dans équation d'équilibre
Trap: 5p−50 = 2050−20p → 5p−20p = 2050−50
Vrai: 5p−50 = 2050−20p → 5p+20p = 2050+50 → 25p=2100
Règle: les p d'un côté, les constantes de l'autre, en CHANGEANT les signes
13. Décision production court terme
Trap: si p < min CM → fermer
Vrai: on ferme seulement si p < min CVM. Entre min CVM et min CM, on produit avec perte
Règle: court terme → comparer à CVM (pas CM). On supporte la perte si on couvre les CV
